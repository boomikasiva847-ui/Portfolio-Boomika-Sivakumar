# Developer Portfolio

A modern, dark, neon‑accented, single‑page developer portfolio built with **HTML5**, **CSS3 (Flexbox & Grid)**, and **vanilla JavaScript**.

## Files

- `index.html` – Main page with all sections (Hero, About, Skills, Projects, Achievements, Contact, Footer)
- `style.css` – Dark futuristic UI, glassmorphism cards, responsive layout, animations
- `script.js` – Smooth scrolling, scroll‑reveal animations, skill progress bars, stat counters, nav behavior, and contact form validation

## Running the site

1. Place all three files in the same folder (already done in this project).
2. Open `index.html` directly in your browser, or serve the folder using a simple static server.

## Customization

- **Name & role**: Update the hero section in `index.html` (`#hero-name`, role text, and intro paragraph).
- **Photo**: Replace the placeholder image URL in the hero section with your own profile image.
- **Skills**: Adjust skill names and percentages in the Skills section; matching progress bar values are in the `data-progress` attributes.
- **Projects**: Edit project titles, descriptions, tech tags, and the GitHub/Live Demo links in the Projects section.
- **Stats**: Change the numbers in the Achievements section via the `data-count` attributes on each stat.
- **Contact**: Update email, phone, location, and social URLs in the Contact section.

## Notes

- The contact form uses **front‑end validation only** and does not send real emails; hook it to a backend or service if needed.
- The layout is fully responsive for mobile, tablet, and desktop, with smooth scroll and subtle scroll‑triggered animations.
